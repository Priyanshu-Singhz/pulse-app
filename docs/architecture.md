# Architecture Overview

## iOS App
- SwiftUI + MVVM
- Combine for reactive state
- URLSession for networking

## Flutter App
- BLoC pattern
- Dio for HTTP
- Firebase for push notifications

## Backend
- FastAPI + PostgreSQL
- JWT authentication
- Deployed on AWS EC2

## Security
- JWT tokens
- HTTPS only

## Infrastructure
- Docker + docker-compose
- GitHub Actions CI/CD
- AWS S3 for media storage
- CloudFront CDN
